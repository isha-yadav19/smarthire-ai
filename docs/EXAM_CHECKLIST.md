## If Something Goes Wrong

### Backend Won't Start
1. Check port 5000 not in use
2. Kill process: `taskkill /F /IM python.exe`
3. Restart: `python simple_api.py`

### Streamlit Won't Start
1. Check port 8501 not in use
2. Clear cache: `streamlit cache clear`
3. Restart: `streamlit run app.py`

### Upload Fails
1. Check `uploads/` folder exists
2. Verify file format (PDF/DOCX)
3. Try with smaller files
4. Show API endpoint instead

### Processing Fails
1. Check job description provided
2. Verify resumes uploaded
3. Show pre-generated results
4. Explain the logic instead

### Demo Completely Fails
1. Stay calm and professional
2. Show code walkthrough
3. Explain architecture
4. Show sample results CSV
5. Answer questions about implementation
